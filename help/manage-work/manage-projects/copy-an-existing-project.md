# 有效率地複製現有專案

影片說明如何複製專案，而不使用範本。 若要&#x200B;複製專案，請開啟所需的專案，按一下專案名稱旁的三點選單，然後選取「複製」。 在「複製專案」視窗&#x200B;中，您可以重新命名專案並變更其狀態，通常為「計畫」，以避免傳送通知給指派的使用者。 您&#x200B;可以選擇是否要複製指派、進度、自訂資料等&#x200B;。
若要完成，請按一下[複製專案]。 &#x200B;
在此過程中，無法直接複製問題，因為這些問題不被視為計畫工作。 若要&#x200B;複製問題，請前往問題標籤，選取所需的問題，並使用「複製到」選項以將其轉移到新專案。 &#x200B;


>[!VIDEO](https://video.tv.adobe.com/v/3456037/?quality=12&learn=on&enablevpops=1)

## 重點提要

* **複製專案：**&#x200B;開啟專案，按一下專案名稱旁邊的三個點功能表，然後選取[複製]以複製專案。 &#x200B;
* **調整專案設定：**&#x200B;重新命名專案，並將其狀態變更為[計畫]，以避免傳送通知給指派的使用者。
* **選擇性複製：**&#x200B;您可以選擇複製指派、進度或自訂資料。 取消選&#x200B;取這些選項可允許在新專案中進行自訂。
* **處理自訂資料：**&#x200B;如果取消選取自訂資料，則會複製不含其資料的表單。 保留&#x200B;核取方塊，以複製表單及其資料。 &#x200B;
* **單獨複製問題：**&#x200B;專案復製程式期間不會複製問題。 若要&#x200B;複製，請前往問題標籤，選取所需的問題，並使用「複製到」選項將其轉移至新專案。 &#x200B;

# 設定匯率

影片提供在Workfront財務追蹤系統中管理貨幣的指南。 它說&#x200B;明，預設貨幣是美元，但其他貨幣可以設定為全域或個別專案和工作角色。
影片強調系統不會提供目前的匯率，因此使用者必須從外部取得匯率。 設定&#x200B;後，匯率成為專案或工作角色的預設值，但不會自動更新；具有適當許可權的使用者必須手動更新。 設定貨幣&#x200B;後，使用者可以建立新專案，並從可用的貨幣選項中進行選取，同時能夠自訂個別專案的匯率。 &#x200B;

>[!VIDEO](https://video.tv.adobe.com/v/3457693/?quality=12&learn=on&enablevpops=1)

## 重點提要

* **預設貨幣：**&#x200B;系統的預設貨幣是美元，但此預設貨幣可全域變更，或針對個別專案和工作角色變更。 &#x200B;
* **新增貨幣：**&#x200B;可在設定區域中設定其他貨幣及其匯率。 &#x200B;
* **手動匯率更新：**&#x200B;系統未提供目前的匯率；使用者必須從外部取得匯率，然後手動輸入。 &#x200B;
* **特定專案自訂：**&#x200B;設定區域中所設定的匯率會作為預設值，但使用者可以自訂個別專案的匯率。 &#x200B;
* **更新與許可權：**&#x200B;設定區域中的變更不會自動更新現有的專案。 更新專案或工作角色的匯率需要適當的使用者許可權。

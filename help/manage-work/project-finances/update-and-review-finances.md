# 更新使用者端專案的財務

此影片提供追蹤時間、費用和建立使用者端專案計費記錄的詳細指南。 關&#x200B;鍵包括：

**追蹤時間和費用：**
追蹤處理專案所花的時間，以及向客戶帳單的專案費用。 &#x200B;
確保使用者在其設定檔或工作角色中設定了成本和計費率。 &#x200B;

**任務設定：**
將適當的成本與收入型態指派給作業。 &#x200B;
成本型別包括固定小時、使用者小時（根據使用者設定檔）和角色小時（根據任務擁有者的工作角色）。

**費用管理：**
透過「費用」索引標籤建立專案層級的費用。 &#x200B;
將費用指派給專案或任務，將其標籤為可記帳或可償還。 &#x200B;

**付費記錄：**
在專案層級建立付費記錄，以追蹤可付費專案（時數、費用、收入）。 &#x200B;
將可記帳時數和費用新增到記帳記錄。
向客戶開立商業發票後，更新帳單記錄狀態和帳單日期。 &#x200B;

**最終備註：**
Workfront會追蹤可記帳專案，但不處理實際帳單；使用者必須使用自己的帳單系統。 &#x200B;

此影片強調正確設定和追蹤以簡化使用者端計費流程的重要性。 &#x200B;

## 重點提要


* **設定成本和計費率：**&#x200B;確保使用者或工作角色在其設定檔中定義成本和計費率，以準確追蹤專案費用。 &#x200B;
* **將成本和收入型別指派給任務：**&#x200B;任務應具有適當的成本和收入型別（例如，固定小時、使用者小時、角色小時），以正確計算可記帳金額。 &#x200B;
* **管理專案費用：**&#x200B;使用[費用]索引標籤來建立費用並將費用分類為可記帳或可償還，以確保追蹤準確。 &#x200B;
* **建立並更新付費記錄：**&#x200B;付費記錄有助於追蹤可付費時數、費用及收入。 新&#x200B;增專案至記錄、更新狀態，並在開立商業發票後記錄記帳日期。 &#x200B;
* **使用外部帳單系統：** Workfront會追蹤可開立帳單的專案，但不會處理實際帳單；使用者必須依賴自己的帳單系統，才能向客戶開立發票。 &#x200B;

# 建立職位角色和費率卡

影片說明如何建立工作角色和費率卡，以及如何從專案附加和移除費率卡。


>[!VIDEO](https://video.tv.adobe.com/v/3486517/?quality=12&learn=on&enablevpops=1)

## 重點提要

* **工作角色結構專案任務：**&#x200B;工作角色代表任務所需的技能，可以根據使用者的能力和可用性指派給使用者。
* **費率卡定義收費率：**&#x200B;費率卡會列出工作角色的收費率，並做為專案收費的客戶合約。
* **彈性費率管理：**&#x200B;職務角色和費率卡都允許設定有效日期的成本和帳單費率，支援以時間為基礎的定價變更。
* **費率卡覆寫：**&#x200B;將費率卡附加至專案時，其費率會覆寫工作角色費率，但專案特定的覆寫不受影響。
* **簡單的附件和更新：**&#x200B;可從專案中附加或移除費率卡，而且可將更新推送至現有專案以確保帳單準確。

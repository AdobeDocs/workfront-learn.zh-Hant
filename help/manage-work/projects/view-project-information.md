# 探索專案管理的篩選器、檢視和群組

本影片提供Workfront中篩選器、檢視和分組的概觀，以協助使用者有效率地管理和分析專案、任務和問題。 關&#x200B;鍵包括：

* **專案篩選器：**&#x200B;篩選器可自訂以縮小專案清單。 例如&#x200B;「我擁有的專案」、「我參與的專案」、「目前」和「落後於預定計畫」。 如「有風險」和「落後於排程」等&#x200B;篩選器會詳細說明，顯示它們如何依賴不同欄位。
* **專案檢視：**&#x200B;檢視會顯示各種專案詳細資料，例如計畫、預計和實際開始/完成日期。 例如&#x200B;「標準檢視」、「所有日期檢視」和「真實劇本檢視」，可提供條件更新和摘要。 &#x200B;
* **專案群組：**&#x200B;專案可依進度狀態或完成日期等欄位依月份分組，以便進行更好的組織和分析。 &#x200B;
* **任務篩選器：**&#x200B;篩選器可協助識別與特定角色相關的未指派任務或任務。 清除&#x200B;篩選器會顯示父系任務結構，而排序可依任務編號調整。 &#x200B;
* **任務檢視：**&#x200B;類似於「狀態檢視」的檢視會顯示進度、評論、檔案、問題、核准、關鍵路徑和里程碑的任務狀態和旗標。
* **任務群組：**&#x200B;與專案群組類似，任務可以依進度狀態等欄位分組。 &#x200B;
* **問題：**&#x200B;問題的篩選器、檢視和群組類似於專案和任務的篩選器、檢視和群組。 熱門&#x200B;檢視包括「登入資訊」和「主要聯絡人」等分組有助於組織問題。 &#x200B;

影片強調自訂，並提供教學課程的連結，以供您進一步學習。 &#x200B;

## 重點提要

* **可自訂的篩選器：**&#x200B;如「我擁有的專案」和「進度落後」等篩選器有助於縮小專案清單的範圍，使用者可以自訂或棧疊篩選器以取得更具體的結果。 &#x200B;
* **詳細檢視：**&#x200B;檢視（例如「所有日期檢視」和「真實內文檢視」）可提供計畫、預計和實際日期的深入分析，以及專案摘要和條件更新。 &#x200B;
* **有效群組：**&#x200B;專案與任務可依進度狀態或完成日期等欄位分組，以更妥善地進行組織和分析。 &#x200B;
* **任務管理：**&#x200B;任務篩選器和檢視（例如「狀態檢視」）可協助識別未指派的任務、追蹤進度以及存取相關詳細資訊，例如註解、檔案和里程碑。 &#x200B;
* **問題追蹤：**&#x200B;篩選器、檢視和問題群組，例如「登入資訊」和「主要連絡人」，可簡化問題管理，並提供提交時間表和連絡人的清晰度。 &#x200B;

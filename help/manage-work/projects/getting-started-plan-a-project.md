# 在Workfront中有效計畫專案

影片提供如何在Workfront中有效規劃專案的指引。 它&#x200B;強調奠定堅實基礎的重要性，以確保專案按時完成並取得成功。 關&#x200B;鍵步驟包括瞭解五個關鍵欄位：任務、指派、持續時間、計畫時數和前置任務。 這&#x200B;些欄位可讓您快速且準確地調整專案計畫。 &#x200B;

影片強調真實的時間表、團隊可用性和準確規劃的重要性，為專案成功奠定堅實的基礎。 &#x200B;

>[!VIDEO](https://video.tv.adobe.com/v/335086/?quality=12&learn=on&enablevpops=1)

## 重點提要

* **專注於五個關鍵欄位：**&#x200B;瞭解任務、指派、期間、計畫時數及前置任務，以便對您的專案計畫進行有效率且精確的變更。 &#x200B;
* **將專案狀態設定為[計畫]：**&#x200B;請確認專案狀態設定為[計畫]，以防止在調整計畫時收到不必要的通知。 &#x200B;
* **使用適當的檢視：**&#x200B;利用標準檢視或自訂檢視，將重點放在基本專案詳細資訊，例如工作、持續時間和指派。 &#x200B;
* **選擇實際的開始日期：**&#x200B;設定實際的開始日期，以符合團隊可用性並避免不實際的情境，例如假日或週末。 &#x200B;
* **利用排程模式：**&#x200B;使用「開始日期」模式計算完成日期，或使用「完成日期」模式計算開始日期，確保時間表符合專案需求。 &#x200B;

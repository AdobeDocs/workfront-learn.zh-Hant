# 使用專案量度追蹤工作進度

影片說明如何使用專案量度追蹤專案進度。 它&#x200B;會示範如何存取專案左側面板中的「量度」索引標籤，其中可能需要捲動並按一下「顯示更多」。 頂端的KPI區域會顯示關鍵績效指標，例如已完成、未完成及逾期的工作。 使用&#x200B;者可以按一下指示器來檢視特定工作，並使用圖表來按狀態或優先順序將工作或問題分組。 將游標暫留在欄&#x200B;上可顯示數量，而圖表則顯示完成與未完成任務或問題的比率。 量&#x200B;度也可以根據任務的狀態（完成、未完成、即將發生或過期）依受指派人顯示任務。 系統會儲存檢視量度的偏好設&#x200B;定，以供日後的造訪使用。 最&#x200B;後，影片會說明如何使用「匯出」按鈕，將度量頁面匯出為PNG檔案。 


>[!VIDEO](https://video.tv.adobe.com/v/336667/?quality=12&learn=on&enablevpops=1)

## 重點提要

* **存取專案量度：**&#x200B;導覽至專案左側面板中的量度索引標籤，以追蹤進度。 您&#x200B;可能需要向下捲動並按一下「顯示更多」才能找到它。 
* **關鍵績效指標(KPI)：** KPI區域顯示已完成、未完成及逾期的工作。 按一&#x200B;下指示器可讓您檢視特定工作。 
* **任務和問題圖表：**&#x200B;按狀態或優先順序將任務或問題分組，顯示完成與未完成的比率。 將游標暫留在欄&#x200B;上會顯示數量。 
* **依受指派人的量度：**&#x200B;依受指派人根據其狀態（完成、未完成、即將進行或逾期）檢視工作。 會&#x200B;儲存偏好設定以供日後的造訪使用。 
* **匯出量度：**&#x200B;按一下畫面右上角的[匯出]按鈕，將量度頁面匯出為PNG檔案。 &#x200B;

# 在Workfront中建立請求佇列

影片提供在Workfont中建立請求佇列的逐步指南。 影片的&#x200B;結尾會示範如何使用新建立的佇列提交請求、示範在「已提交請求」索引標籤中輸入詳細資訊及驗證提交的程式。 

>[!VIDEO](https://video.tv.adobe.com/v/335221/?quality=12&learn=on&enablevpops=1)

## 重點提要

* **建立請求佇列：**&#x200B;開始先建立新專案並命名它，稍後再做為請求佇列。 請&#x200B;確定勾選「發佈為說明請求佇列」方塊，將其正式指定為佇列。 
* **存取許可權：**&#x200B;從四個選項中選擇可以提交請求至佇列的人員：任何人、專案檢視者、公司成員或群組成員。 &#x200B;
* **請求型別：**&#x200B;自訂佇列以包含特定型別的請求，例如問題、錯誤報告、變更順序或一般請求。 預&#x200B;設選項為「問題」。 
* **可自訂欄位：**&#x200B;設定預設回應時間、許可權和其他佇列特性等其他詳細資料，以調整其功能。 &#x200B;
* **讓佇列處於即時狀態：**&#x200B;儲存設定並將專案狀態設為「目前」以啟動佇列。 請&#x200B;求提交並在「請求已提交」標籤中驗證，以測試其功能。

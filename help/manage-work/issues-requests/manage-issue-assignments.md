# 有效管理問題指派

管理問題是專案經理人和被指派解決問題的人員的重要職責。

觀看這段影片，您將瞭解如何：

* 指派問題
* 尋找問題指派
* 更新問題的處理進度
* 更新問題狀態

>[!VIDEO](https://video.tv.adobe.com/v/3419931/?quality=12&learn=on&enablevpops=1)

## 重點提要

* **指派問題：**&#x200B;使用指派欄位中的內嵌編輯，將問題指派給使用者或團隊，並確保在完成指派後按一下「儲存」。 團&#x200B;隊領導者可以使用工作負載平衡器透過將任務拖放到團隊成員的行事曆來指派任務。 
* **尋找指派的問題：**&#x200B;指派給您的問題出現在首頁工作清單中，此清單可供篩選。 團隊&#x200B;指派的問題會顯示在團隊請求區段中，團隊成員可以接受或重新指派任務。 這&#x200B;些問題也會出現在工作負載平衡器中，以提升可見度。 
* **更新進度：**&#x200B;發佈更新並記錄問題的時數，以提供工作進度的可見度。 使&#x200B;用「At name」功能可將註解導向特定使用者或團隊。 會傳送通知&#x200B;以聽取註解，但不會傳送記錄時數或狀態變更。 
* **變更問題狀態：**&#x200B;更新問題狀態以反映進度，例如「處理它」或「完成」。 未解決&#x200B;的問題會導致相關任務和專案無法標籤為完成。 
* **特定團隊的工作流程：**&#x200B;遵循您組織的特定工作流程和系統設定，以有效管理問題。 &#x200B;

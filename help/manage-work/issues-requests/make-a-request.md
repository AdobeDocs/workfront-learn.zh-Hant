# 提出要求

在許多組織中，開始新專案、開發新產品或建立交付專案的第一步是提出請求 [!DNL Workfront]. 您可以追蹤請求的進度，並在需要時提供其他資訊。

在本影片中，您將瞭解如何：

* 導覽至請求區域
* 提出要求
* 檢視已提交的請求
* 尋找請求的草稿

# 提出請求

在許多組織中，啟動新項目、開發新產品或建立交付項的第一步是在 [!DNL Workfront]. 您可以追蹤要求的進度，並視需要提供其他資訊。

在此影片中，您將學習如何：

* 導覽至請求區域
* 提出請求
* 查看提交的請求
* 查找請求草稿

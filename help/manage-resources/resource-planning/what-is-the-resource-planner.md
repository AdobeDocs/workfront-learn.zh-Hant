# 什麼是[!DNL Resource Planner]？

要真正賦予人員足夠的能力，管理者必須規劃人員的工作並進行排程。然而，要物色擁有適當技能而且在適當的時間有空承接工作的人員，並同時每月、每季甚至每年擷取這些資料並進行管理，是非常龐大的工作。

不過，使用 Workfront 的資源規劃工具可以讓資源相關的任務變得輕鬆一些。

# 什麼是 [!DNL Resource Planner]？

若要真正增強員工的能力，經理必須規劃並排程員工的工作。 然而，在適當的時間尋找具備適當技能的人員，同時嘗試擷取這些資料並每月、每季甚至每年管理這些資料，是一項主要工作。

不過，使用Workfront的資源規劃工具可讓這項工作輕鬆一點。

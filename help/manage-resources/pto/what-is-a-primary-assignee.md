# 什麼是主要受託人？

將兩個或多個用戶分配給任務時，主受託人存在。 主要受託人的日曆時間是Workfront在套用「使用者時間關閉」設定時要查看的時間。

從專案頁面找出主要受託人。 按一下「工作總攬」(Assignments)欄位，然後按一下拐角中的高級工作總攬表徵圖。

![多個受分配者](assets/pa_01.png)

在「高級分配」窗口中，主要受分配人或任務責任人將使用綠色複選標籤指定。 將滑鼠指標暫留在其他使用者上，然後按一下「設為主要」連結以變更任務擁有者。

![選取主要受託人](assets/pa_02.png)

您也可以從任務頁調整主要受託人。 勾選專案清單中任務的方塊，然後按一下「編輯」圖示。 捲動到「工作總攬」(Assignments)部分，然後按一下按鈕以指定任務責任人。

![任務責任人按鈕](assets/pa_03.png)

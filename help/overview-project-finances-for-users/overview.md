# 使用者專案財務概觀

瞭解如何追蹤專案的財務資訊、設定成本和收入型別，以及覆寫收費率。

Workfront的用途並非取代您目前使用的任何財務追蹤或計費軟體，但可提供貨幣追蹤和報表功能。

>[!PREREQUISITES]
>
>* 對使用Workfront的基本瞭解
>* 有權編輯財務資料的標準授權型別

## 目標客群

想要在Workfront中追蹤財務的使用者。

# 使用者專案財務概觀

了解如何追蹤專案的財務資訊、設定成本和收入類型，以及覆寫計費費率。 本教學課程是專為將追蹤與專案相關財務資料的使用者所設計。

Adobe Workfront的用途並非取代您目前使用的任何財務追蹤或計費軟體，但可提供貨幣追蹤和報表功能。

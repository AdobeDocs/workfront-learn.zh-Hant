# 提出請求

在許多組織中，啟動新專案、開發新產品或建立可交付成果的第一步是在 Workfront 中提出請求。 您可以追蹤請求的進度並在必要時提供其他資訊。

觀看這段影片，您將瞭解如何：

* 導覽至請求區域
* 提出請求
* 檢視已提交的請求
* 複製請求
* 尋找請求的草稿
* 查看最後三個請求路徑

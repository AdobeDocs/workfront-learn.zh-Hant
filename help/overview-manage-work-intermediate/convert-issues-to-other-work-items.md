# 將問題/請求轉換為任務

在本影片中，您將學習如何將問題或請求轉換為任務。

>[!VIDEO](https://video.tv.adobe.com/v/3427605/?quality=12&learn=on&enablevpops=1)

>[!NOTE]
>
>專案時間表上不能新增問題，因為問題代表「未規劃的工作」。專案時間表適用於「已規劃的工作」，即任務。

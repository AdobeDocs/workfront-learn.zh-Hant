# 定義不同的使用案例並尋找最佳方法

在建置之前，在考慮自動化時，您可以部署一些策略，以盡量減少需要的返工或必須經過的迭代次數。

您可以定義由特定事件觸發的清楚使用案例，以節省最多時間。 如果你不及早確定自己正朝著正確的方向前進，那麼最終會出現複雜或低效的情況。

在此影片中，您將學習如何：

* 識別您的案例的特定使用案例
* 選擇正確的觸發器模組
* 為高效能確定正確的設計
* 挑戰假設，向利益相關方提供最佳建議

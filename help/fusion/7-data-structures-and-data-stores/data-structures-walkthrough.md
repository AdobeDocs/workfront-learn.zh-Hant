# 資料結構操作示範

開啟含有時間記錄清單的 CSV 檔案。 這些時間記錄是多位使用者在某幾天內記錄的分鐘數。 目標是擷取這些資訊並產生新的 CSV 檔，顯示每位使用者每天記錄的總時數 (以小時為單位)。

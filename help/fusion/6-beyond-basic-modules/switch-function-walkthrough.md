# 交換機功能逐步說明

## 總覽

對於簡單的資料更改，請使用Switch函式將模組欄位內的一個值轉換為另一個值。 在本練習中，將兩個字母的鍵更改為電子郵件中發送的項目進度狀態的實際名稱。

# 尋找和管理請求

觀看這段影片，您將瞭解如何尋找指派給您的請求、您建立的請求以及您管理的請求。

您將瞭解如何在下列位置尋找這些請求：

* [!UICONTROL Workfront 首頁]
* 「[!UICONTROL 請求]」區域
* 「[!UICONTROL 團隊]」頁面
* 在自訂儀表板中

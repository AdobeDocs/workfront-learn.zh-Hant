# 建立多階段校訂工作流程

校訂工作流程可能很簡單，只需幾個檢閱者和一個截止日期。 或者，它們可能更複雜，有多個截止日期，而且人們會以特定順序檢視校訂。

[!DNL Workfront] 有兩種校訂工作流程：基本和自動化，有時稱為進階。

自動化工作流程具有多個階段或步驟，需要按特定順序完成。 除了自己的截止日期外，每個階段都有自己的收件者集（稽核者和核准者）。

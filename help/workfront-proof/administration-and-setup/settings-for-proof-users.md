# 校訂使用者的設定

開始在中建立使用者之前 [!DNL  Workfront]，最佳實務是讓您的全域 [!DNL Workfront] 和校訂系統設定完成。 其中一些設定會在建立時套用至使用者設定檔。 但如果您在建立部分使用者之前並未完成這些設定，請不要擔心……您可以在每個個別的使用者設定檔上編輯這些設定。


在本影片中，您將瞭解如何：

* 決定哪些使用者取得校訂授權
* 調整校樣使用者設定 [!DNL  Workfront]
* 在校訂後端設定中調整校訂使用者設定

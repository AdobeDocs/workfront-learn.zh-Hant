# 使用建立版面範本 [!DNL Workfront Goals]

在 **[!UICONTROL 設定]** 區域，您也可以建立並指派版面範本，其中包含 [!DNL Workfront Goals] 以便您和您的使用者能更輕鬆存取。

在此影片中，您將學習如何：

* 建立版面範本
* 將版面範本指派給使用者
* 新增 [!DNL Goals] 至版面範本
* 變更 [!DNL Goals] 適合貴組織的術語
* 新增 [!DNL Goals] 到 [!DNL Workfront] [!UICONTROL 主菜單]

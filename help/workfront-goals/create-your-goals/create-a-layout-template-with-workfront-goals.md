# 建立版面範本 [!DNL Workfront Goals] 使用者

在 **[!UICONTROL 設定]** 區域，您也可以建立並指派配置範本，包括 [!DNL Workfront Goals] 讓您和您的使用者更輕鬆地存取它。

在本影片中，您將瞭解如何：

* 建立版面範本
* 將版面配置範本指派給使用者
* 新增 [!DNL Goals] 至版面範本
* 變更 [!DNL Goals] 適合您組織的術語
* 新增 [!DNL Goals] 至 [!DNL Workfront] [!UICONTROL 主要功能表]

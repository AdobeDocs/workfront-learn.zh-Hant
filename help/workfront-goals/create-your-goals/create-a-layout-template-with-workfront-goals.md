# 為 [!DNL Workfront Goals] 使用者建立版面範本

在「**[!UICONTROL 設定]**」區域中，您可以建立並指派包含 [!DNL Workfront Goals] 的版面範本，讓您和使用者存取更方便。

觀看這段影片，您將瞭解如何：

* 建立版面範本
* 指派版面範本給使用者
* 新增 [!DNL Goals] 到版面範本
* 變更 [!DNL Goals] 術語以符合貴組織需求
* 把 [!DNL Goals] 新增到 [!DNL Workfront] [!UICONTROL 主選單]

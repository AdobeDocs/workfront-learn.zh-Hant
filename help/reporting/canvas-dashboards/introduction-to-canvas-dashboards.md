# Canvas儀表板簡介

「畫布」儀表板功能是在Workfront中建立儀表板的另一種方式。 此功能目前在一般Beta中，這表示所有Workfront客戶都可以使用此功能（如果他們已註冊測試版）。 系統管理員需要執行註冊。 本影片將展示如何進行。

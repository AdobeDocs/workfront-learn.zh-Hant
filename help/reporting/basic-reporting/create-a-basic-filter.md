# 建立基本篩選器

此影片說明如何在Workfront中建立和使用篩選器，以根據特定條件自訂清單報表。 篩選&#x200B;器可讓使用者藉由定義條件來縮小清單（例如任務、專案或檔案）的範圍。 只有符&#x200B;合篩選條件且登入使用者可存取的專案才會出現在清單中。 &#x200B;


>[!VIDEO](https://video.tv.adobe.com/v/335151/?quality=12&learn=on&enablevpops=1)

## 重點提要

* **篩選器自訂清單報告：** Workfront中的篩選器可讓使用者縮小清單（任務、專案、檔案等） 根據特定條件，僅顯示符合條件且登入使用者可存取的專案。 &#x200B;
* **常見篩選型別：**&#x200B;範例包含使用者篩選器，例如「我的專案」（顯示登入使用者特定的專案）和日期篩選器，例如「本月到期」（顯示具有當月內計畫完成日期的專案）。 &#x200B;
* **建置篩選器：**&#x200B;篩選器是使用規則建立的，例如「AND」篩選器（所有條件都必須為true）或「OR」篩選器（任何條件都可以為true）。 例&#x200B;如，「我的已完成任務」篩選器結合已完成任務的規則以及指派給登入使用者的任務。 &#x200B;
* **共用和移除篩選器：**&#x200B;篩選器可以與其他Workfront使用者共用，或是透過篩選器功能表移除，使其成為彈性和合作工具。 &#x200B;
* **清除命名慣例：**&#x200B;在篩選名稱中使用「My」或「I」可協助使用者瞭解篩選結果是否特定於自己，進而提高清晰度和可用性。 &#x200B;

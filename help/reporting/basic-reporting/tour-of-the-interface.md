# 了解行事曆報告介面

行事曆報告是動態報告，在行事曆檢視中顯示工作和休假時間。您可以在行事曆報告中顯示以下物件的日期資訊：

* 任務
* 專案
* 問題
* 休假

觀看這段影片，您將會了解以下資訊：

* 行事曆分組
* 行事曆視圖
* 行事曆動作選單
* 摘要資訊
* 以及如何建立臨時的行事曆項目

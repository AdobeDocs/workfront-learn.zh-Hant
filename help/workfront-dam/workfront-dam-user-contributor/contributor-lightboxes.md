## 資料夾與燈箱

資料夾可用來儲存和組織資產。 資料夾可授予資產的存取權。 中可能有更多資料夾 [!UICONTROL Workfront DAM] 比你看的還多。

燈箱是 [!UICONTROL Workfront DAM] 您可以用來收集和組織您需要的資產。 您可以為自己建立單一燈箱，或為您進行的每個指派建立燈箱。 您只會看到您建立的燈箱，除非其他使用者與您共用燈箱或要求您協作使用燈箱。
